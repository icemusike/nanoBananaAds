## Color Application Guidelines

### When to Use Each Scheme

**B2B/Enterprise** → Trust Builder, Tech Blue, Minimal Mono
**E-commerce** → Action Orange, Bold Red, Energetic Yellow
**Luxury/Premium** → Premium Purple, Minimal Mono, Warm Sunset
**Health/Wellness** → Growth Green, Fresh Mint
**Tech/Innovation** → Tech Blue, Minimal Mono
**Creative Services** → Warm Sunset, Premium Purple, Minimal Mono
**Sales/Urgency** → Bold Red, Action Orange

### Customization Tips

1. **Keep brand colors**: Replace accent/CTA colors with brand colors
2. **Test contrast**: Ensure all text passes WCAG AA standards (4.5:1)
3. **Limit colors**: Use maximum 3-4 colors from palette
4. **Maintain hierarchy**: Primary for backgrounds, accent for CTAs
5. **Consider psychology**: Match color emotions to product benefits

### Quick Substitution Formula

User provides brand color? Here's how to adapt:

```
IF user_brand_color is:
  Blue → Use Tech Blue scheme
  Red → Use Bold Red scheme
  Green → Use Growth Green scheme
  Purple → Use Premium Purple scheme
  Orange → Use Action Orange scheme
  
THEN:
  Replace "accent" with user_brand_color
  Replace "ctaPrimary" with user_brand_color
  Keep backgrounds neutral (white/light gray)
  Ensure CTA contrasts with background
```

## Tailwind CSS Color Configuration

To use these schemes in Tailwind, add to `tailwind.config.js`:

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        // Example: Trust Builder scheme
        'brand': {
          50: '#EFF6FF',  // Lightest
          100: '#DBEAFE',
          200: '#BFDBFE',
          300: '#93C5FD',
          400: '#60A5FA',
          500: '#3B82F6',
          600: '#1E40AF', // Primary
          700: '#1E3A8A',
          800: '#1E3A8A',
          900: '#1E3A8A', // Darkest
        },
        'accent': {
          500: '#F59E0B',  // CTA color
        }
      }
    }
  }
}
```

Use in HTML as: `bg-brand-600`, `text-accent-500`, etc.
